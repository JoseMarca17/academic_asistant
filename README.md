# Asistente Académico Personal

Asistente basado en IA que analiza apuntes en Markdown y PDF
para responder preguntas y generar resumenes usando embeddings.

## Funcionalidades
- Lectura de notas Obsidian
- Búsqueda semántica
- Respuestas basadas en apuntes propios

## Instalación
```bash
pip install -r requirements.txt
